    Δ ∙ \fresh{τ} ⊢ ν ∋ r ⊠ Θ ∙ \stale{τ}
  \end{array}}
}{Γ ⊢ \cas{t}{ν}{l}{r} ∈ ν ⊠ Θ

}{Γ ⊢ \prr{t} ∈ τ ⊠ Δ

 {Γ ⊢ t ∈ 𝟘 ⊠ Δ
}{Γ ⊢ \exf{σ}{t} ∈ σ ⊠ Δ
}

 {Γ ⊢ σ ∋ t ⊠ Δ
}{Γ ⊢ \cut{t}{σ} ∈ σ ⊠ Δ
